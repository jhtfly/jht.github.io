# 蒋浩涛的个人博客

使用 Jekyll 搭建，部署在 GitHub Pages。

## 功能
- 技术文章发布（支持 Markdown）
- 提示词库（一键复制/下载，密码保护）
- Skill 技能文件（分类下载，密码保护）
- 关于页（个人信息 + 资助渠道）

## 部署
详见 [deploy-guide.md](deploy-guide.md)

## 目录结构
```
_config.yml        # 站点配置
_data/             # 数据文件（提示词、Skill 数据）
_downloads/        # 可下载文件
_includes/         # 可复用组件
_layouts/          # 页面模板
_pages/            # 独立页面（关于、提示词、Skills）
_posts/            # 博客文章（Markdown）
assets/            # CSS、JS、图片
```
