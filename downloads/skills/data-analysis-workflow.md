## 描述
完整的端到端数据分析流程技能，从数据获取到最终报告生成。

## 适用场景
- 新数据集的系统性分析
- 日常数据分析任务的标准化流程
- 数据质量检查和预处理

## 工作流程

### 1. 数据获取
- 使用 pandas 读取 CSV/Excel/数据库
- 使用 Baostock API 获取金融数据
- 数据格式验证

### 2. 数据清洗
- 缺失值处理（删除/填充策略）
- 异常值识别与处理
- 数据类型转换
- 重复值处理

### 3. 数据分析
- 描述性统计分析
- 相关性分析
- 趋势分析
- 分组对比分析

### 4. 数据可视化
- 趋势图、柱状图、散点图
- 热力图、箱线图
- 配色统一、中文标签

### 5. 报告生成
- 使用 python-docx 生成 Word 报告
- 标准格式：标题 → 摘要 → 正文 → 结论
- 图表嵌入文档
