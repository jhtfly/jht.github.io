# Skill: Baostock 数据获取

## 基本信息
- 名称：baostock-api
- 版本：2.0
- 作者：蒋浩涛
- 最后更新：2026-05-20

## 描述
使用 Baostock API 获取 A 股全量数据，包括 K 线、财务报表、宏观数据等。已覆盖全部 23 个 API 接口。

## 适用场景
- A 股历史行情数据获取
- 上市公司财务数据下载
- 宏观经济指标查询
- 量化回测数据准备

## API 接口覆盖
1. 登录/登出
2. 日线数据查询
3. 周线数据查询
4. 月线数据查询
5. 分钟线数据查询
6. 复权因子查询
7. 季度财务报表
8. 利润表
9. 资产负债表
10. 现金流量表
...（共23个接口）

## 数据规模
- 覆盖股票：5201 只
- 数据量：987 万行
